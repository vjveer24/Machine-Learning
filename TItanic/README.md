Titanic Data Analysis Project


A data science project analyzing the Titanic passenger dataset to discover factors that influenced survival rates. 
This project was built to master core Python data libraries.


Skills Learned

NumPy: Multi-dimensional arrays, data manipulation, and mathematical operations.

Pandas: Data cleaning, handling missing values, filtering, and aggregation.

Matplotlib: Creating basic plots, customizing figures, labels, and legends.

Seaborn: Advanced statistical visualizations, heatmaps, and distribution plots.

Tech Stack

Python 3

Jupyter Notebook

Installation & UsageClone this repository.

Install dependencies: pip install numpy pandas matplotlib seaborn.

Open the Jupyter Notebook to view the analysis.
